**SISEKOLAH** is an application with Software as a Service concept

## Excellent Feature

- Multiple Subdomain
- Multiple Database
- Subscription plan
- Coupon / Discount
- Access Control Level

## Tech Stack

- Laravel 9 (PHP8)
- Vue 3 (Typescript)
