# Feature

## Login

We have multiple login form, for landlord using root domain and for tenant using subdomain based on tenant domain with folowing url `tenant.{rootdomain}.com/auth/login`

![Login Form](/img/feature/auth/login.png)

## Register

We are using multiple form step wizard to handle user registration, you can preview your info before complete the registration
