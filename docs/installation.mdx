This guide will guide you through the installation after downloading sisekolah

## Setup

### Extract ZIP file

After downloading the file received via email address, extract it with the following command

```shell
unzip sisekolah.zip
```

Go into project directory

```shell
cd sisekolah
```

### Setup backend dependencies

```shell
composer install
```

### Setup frontend dependencies

```shell
yarn install
```
