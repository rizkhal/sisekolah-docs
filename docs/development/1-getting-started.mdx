The project directory always have 2 top directory

    ├── Core
    │   ├── Admin
    │   └── Client
    └── Global

The `Core` directory is used for Core App functionality, and it separating for `SaaS Admin` and `SaaS Client` by `Admin` and `Client` directory, for `Global` directory is used for Global usage, for example if a controller will used for `Admin` and `Client` it will be placed under the `Global` directory
